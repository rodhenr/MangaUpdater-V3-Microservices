MangaUpdater.Services.AnilistConnector

Purpose: Connector to retrieve a user's manga collection from Anilist.

Required environment variables: none (uses appsettings.json / appsettings.Development.json).

How to run locally:

1. From repository root run:

   dotnet run --project MangaUpdater.Services.AnilistConnector

2. The service uses `appsettings.Development.json` when running in Development environment.
